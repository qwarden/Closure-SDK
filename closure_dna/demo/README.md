# Closure DNA Demos

This folder holds ready-to-open demo databases for the local web UI.

- source data lives in `closure_dna/demo_data`
- built databases live in `closure_dna/demo/databases`

Use:

```bash
python3.13 -m closure_dna build-demo-db all
python3.13 -m closure_dna web
```

Then open a demo database from the built-in database list in the web UI.
